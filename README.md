# ClipForge

ClipForge is a free online tool that allows users to download videos and audio from YouTube and TikTok instantly.

## Features
- Download YouTube videos as MP4
- Download YouTube audio as MP3
- Choose video quality
- Download TikTok videos without watermark
- No registration required
- Free forever

## Tech Stack
- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js + Express
- Video Engine: ytdl-core

## How to Run Locally
1. Install Node.js
2. Navigate to the server folder
3. Run:
   npm install
   node index.js
4. Open client/index.html in your browser

## Deployment
This project is ready to be deployed on Render or any Node.js compatible hosting service.

## Legal
